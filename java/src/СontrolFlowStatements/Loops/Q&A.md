
# 1. Що таке цикл у програмуванні?

- Цикл в програмуванні є структурою, яка дозволяє виконувати блок коду декілька разів. Він дозволяє повторювати
  виконання певного фрагмента коду, поки задана умова є істинною або поки не буде досягнута певна кількість ітерацій.

# 2. Які типи циклів доступні в Java?

В Java доступні наступні типи циклів:
  - `for` - цикл з передумовою, який виконується певну кількість разів заздалегідь відому кількість разів.
  - `while` - цикл з передумовою, який виконується, доки певна умова є істинною.
  - `do-while` - цикл з післяумовою, який виконується принаймні один раз, а потім продовжується, доки певна умова
  є істинною.

# 3. Як використовувати цикл `for` в Java?

- Цикл `for` використовується для повторення фрагмента коду протягом певної кількості ітерацій. 
Синтаксис циклу `for` виглядає наступним чином:
   ```java
   for (ініціалізація; умова; інкремент) {
       // блок коду, що виконується на кожній ітерації
   }
   ```
Значення `ініціалізація` встановлює початкове значення для лічильника циклу. Умова `умова` перевіряється перед 
кожною ітерацією. Якщо умова є істинною, блок коду виконується. Після кожної ітерації виконується `інкремент`, 
який змінює значення лічильника. Цикл `for` виконується до тих пір, поки умова є істинною.

# 4.Як використовувати цикл `while` в Java?
- Цикл `while` використовується для повторення фрагмента коду, поки певна умова є істинною. Синтаксис циклу `while` 
виглядає так:
   ```java
   while (умова) {
       // блок коду, що виконується, поки умова є істинною
   }
   ```
Умова перевіряється перед кожною ітерацією. Якщо умова є істинною, то виконується блок коду. 
Цикл `while` продовжується, доки умова залишається істинною.

# 5. Як використовувати цикл `do-while` в Java?

    - Цикл `do-while` використовується для повторення фрагмента коду принаймні один раз, а потім продовжується, доки певна умова є істинною. Синтаксис циклу `do-while` такий:
   ```java
   do {
       // блок коду, що виконується
   } while (умова);
   ```
Умова перевіряється після кожної ітерації. Цикл `do-while` завжди виконується принаймні один раз, 
навіть якщо умова з першого ж разу є хибною. Після першої ітерації умова перевіряється, і якщо вона є істинною, 
цикл продовжується.

# 6. Як можна припинити виконання циклу раніше, ніж досягнута кінцева умова?

- Виконання циклу можна припинити за допомогою оператора `break`. Коли оператор `break` з'являється в середині 
циклу, виконання циклу негайно припиняється, і керування передається наступному оператору після циклу.

# 7. Як можна перейти до наступної ітерації циклу без виконання поточної ітерації?

- Для переходу до наступної ітерації циклу без виконання поточної ітерації можна використовувати оператор `continue`.
  Коли оператор `continue` зустрічається всередині циклу, виконання поточної ітерації припиняється, і керування
  переходить до наступної ітерації циклу.
